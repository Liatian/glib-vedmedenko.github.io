Fully autonomous bot suite for social media growth & engagement — 2025 Suite de bots entièrement autonome pour la croissance et l'engagement sur les réseaux sociaux — 2025
Designed and built from scratch a complete autonomous social media automation system that grew the @dreemx account to 30,000+ followers on OpenAI's Sora video platform. The system consists of 4 interconnected Python bots orchestrated by a master controller:
  • Activity Bot — AI-powered comment generation using multi-model cascade (Groq, Cerebras, OpenRouter). Analyzes video content via frame extraction (4 frames at 15/30/60/90%), audio transcription (Whisper ASR), and post descriptions to generate contextually appropriate, human-like comments. Supports remix detection with original post analysis, mood classification, per-post reply limits with deduplication, and emoji-only replies for non-English content
  • Leaderboard Bot — Automatically discovers and tracks top creators from Sora's Remixed leaderboard. Classifies users into tiers (TOP/MIDDLE/REGULAR) based on followers and likes. Implements tiered engagement strategies: 3 likes + 3 personalized AI comments for TOP creators, 3 likes + 1 comment for MIDDLE, 1 like + 1 comment for REGULAR. Features blogger memory system, spam detection, and ban list management
  • Remix Bot — Monitors and comments on remixed videos, delegates post commenting to the Activity Bot for consistent engagement
  • Master Bot (v3) — Orchestrates all three bots with 3-browser parallelism, automatic crash recovery, smart scheduling (duration/until/work-sleep modes), and auto-detection of newest bot file versions for hot-reload
Key technical achievements:
  • Multi-model AI fallback architecture: Cerebras → Groq (6 models) → OpenRouter — ensuring 99%+ uptime for comment generation
  • Vision analysis via Groq Llama-4 Scout for video frame understanding
  • Anti-detection: Selenium with custom Chrome profiles, Cloudflare Turnstile bypass, human-like timing and behavior patterns
  • Robust state management with JSON persistence, crash recovery, and cycle resumption
  • Comment quality validation: banned words filtering, length checks, AI-slop detection, humanization post-processing
