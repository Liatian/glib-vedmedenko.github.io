I designed and built a fully autonomous social media automation system that took the @dreemx account from zero to #1 on the Remixed leaderboard and top 9 globally on OpenAI's Sora platform. Using AI language models (Claude, Grok, Gemini, Llama) and computer vision, the system generates human-like engagement at scale — resulting in 28K+ followers, 361K+ likes, 51K+ remixes, and individual videos surpassing 1.2 million views.

The system is fully scalable and can be adapted for any social media platform (TikTok, Instagram, YouTube, X) or any client's growth strategy — only the selectors and navigation need to change.

The system consists of 4 interconnected Python bots:
  • Activity Bot — AI comment generation with multi-model cascade (Groq, Cerebras, OpenRouter). Video frame analysis (4 frames at 15/30/60/90%), Whisper audio transcription, remix detection, mood classification
  • Leaderboard Bot — Auto-discovers top creators from Sora's leaderboard. Tiered engagement: TOP (3 likes + 3 AI comments), MIDDLE (3+1), REGULAR (1+1). Blogger memory, spam detection
  • Remix Bot — Monitors remixed videos, analyzes original context, delegates commenting to Activity Bot
  • Master Bot v3 — Orchestrates all bots: 3-browser parallelism, crash recovery, smart scheduling, hot-reload
Key technical achievements:
  • Multi-model AI fallback (Cerebras → Groq 6 models → OpenRouter) — 99%+ uptime
  • Computer vision via Groq Llama-4 Scout for video frame understanding
  • Anti-detection: custom Chrome profiles, Cloudflare bypass, human-like behavior
  • JSON state persistence with crash recovery and cycle resumption
